# 🛠 NLOTO System
Персональная облачная среда для выполнения JavaScript скриптов.

⚠️ **Внимание:** Данный репозиторий и все его файлы являются приватной системой. Не распространяйте и не делитесь ссылкой.

### 📂 Структура
- `/Scripts/` — Хранилище исполняемых скриптов.
- `/loader.js` — Ядро интерфейса.
- `/index.html` — Скрытый установщик (требует пароль).
