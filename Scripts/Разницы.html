1. Фильтр «Чётные»
  • Формат: мин-макс (например 3-6).
  • Считает количество чётных чисел в билете П1. Пропускает только те, где количество попадает в диапазон включительно.
  • Максимум автоматически равен количеству чисел в П1 (например для 9 чисел — максимум 9).
  • Минимум — 0 (все нечётные).
Пример для 8 чисел: «3-5». Билет [1,3,5,7,9,11,13,15] имеет 0 чётных — отбрасывается. Билет [2,4,6,8,10,12,14,16] имеет 8 чётных — отбрасывается. Билет [1,2,4,7,9,10,14,17] имеет 4 чётных — проходит. 2. Остальные фильтры (только Поле 1) Сумма, разброс, без соседей, секторы — описаны в предыдущих версиях. 3. Разницы Работают всегда: с пулом — из пула, без пула — генерация на лету с проверкой разниц. Без разниц — обычный рандом. 4. Поле 2 Фильтруется только через Исключения П2 и Включения П2. Билетные фильтры не применяются.
